# Food Delivery Delay Prediction

Machine learning project to predict whether a delivery will be:
- On-Time
- At-Risk
- Delayed

## Dataset Features
- Delivery partner age and ratings
- Weather conditions
- Road traffic density
- Distance between restaurant and customer
- Order and pickup time

## Methods Used
- Data preprocessing and feature engineering
- Handling class imbalance using SMOTE and class weighting
- Model comparison: Logistic Regression, Random Forest, KNN, XGBoost
- Hyperparameter tuning

## Tech Stack
Python, Pandas, Scikit-learn, XGBoost, Matplotlib
